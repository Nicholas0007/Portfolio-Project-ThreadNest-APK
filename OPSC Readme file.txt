Step 1: Get the Source Code
```bash
# Clone the repository
git clone https://github.com/yourusername/StyleSwap.git

# Or download as ZIP and extract
Step 2: Open in Android Studio
Open Android Studio

Click "Open an existing project"

Select the StyleSwap folder

Wait for Gradle to sync and download dependencies

Step 3: Connect Your Phone
On your Android phone, go to Settings > About Phone

Tap "Build Number" 7 times to enable Developer Options

Go back to Settings > Developer Options

Enable "USB Debugging"

Connect your phone to computer via USB cable

Accept the USB debugging prompt on your phone

Step 4: Build and Run
In Android Studio, click the "Run" button (green play icon)

Select your connected device from the list

Click "OK" - the app will build and install automatically

The app will open on your phone once installed

Method 3: Using Android Studio Emulator
Step 1: Create Virtual Device
In Android Studio, click "Device Manager"

Click "Create Device"

Select a phone model (e.g., Pixel 6)

Choose a system image (Android API 34 recommended)

Complete the setup wizard

Step 2: Run on Emulator
Click the "Run" button

Select your virtual device from the list

Click "OK" - the emulator will start and install the app
